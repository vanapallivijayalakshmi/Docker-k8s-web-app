# Docker Kubernetes Web App

## Project Description
This project demonstrates how to deploy a containerized web application using Docker and Kubernetes.

## Tools Used
- Docker
- Kubernetes
- Minikube
- GitHub

## Steps Performed
1. Created a simple HTML web application.
2. Containerized the application using Docker.
3. Created Kubernetes deployment.yaml file.
4. Created Kubernetes service.yaml file.
5. Deployed the application using kubectl.
6. Accessed the application using Minikube service.
